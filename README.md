# Lernreise Support

Mini static support site for App Store Connect.

## App Store Connect URLs

After GitHub Pages is enabled, use:

- Support URL: `https://<github-user>.github.io/lernreise-support/`
- Privacy Policy URL: `https://<github-user>.github.io/lernreise-support/privacy.html`

## Files

- `index.html` - public support page
- `privacy.html` - privacy policy
- `terms.html` - basic terms
- `styles.css` - responsive styling
